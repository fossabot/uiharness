# UIHarness (React)

## Setup

```bash
yarn add -D @uiharness/react
```

Add scripts to `package.json`:

```json
{
  "scripts": {
    "start": "uiharness start",
    "dist": "uiharness dist",
    "serve": "uiharness serve"
  }
}
```
