# create-uiharness

A simple UIHarness module bootsrapper.

Create a module using the UIHarness with a single command.

https://uiharness.com


## Usage

```bash
yarn create uiharness
```

Then follow the command-line prompts to generate the new module.  From there:

```bash
cd <new-module>
yarn
yarn start
```
