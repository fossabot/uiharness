# UIHarness
https://uiharness.com

## Quick Start
```bash
yarn create uiharness
```

Then follow the command-line prompts to generate your new module.

## Manual Setup
```bash
yarn add -D @uiharness/parcel
```
See documentation in [@uiharness/parcel](libs/parcel).
